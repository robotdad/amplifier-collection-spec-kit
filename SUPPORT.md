# Support

## Getting Help

### Documentation

- **README**: Start with the [README.md](README.md) for installation and quick start
- **Issues**: Check [existing issues](https://github.com/robotdad/amplifier-collection-spec-kit/issues) for common questions
- **Discussions**: Join [GitHub Discussions](https://github.com/robotdad/amplifier-collection-spec-kit/discussions) for community help

### Reporting Issues

If you encounter problems:

1. **Search existing issues**: Check if already reported
2. **Create new issue**: Provide:
   - What you were trying to do
   - What happened (actual behavior)
   - What you expected (expected behavior)
   - Version information (amplifier version, collection version)
   - Relevant logs or error messages

### Feature Requests

We welcome feature requests! Please:

1. Check existing issues/discussions
2. Create a new issue with "Feature Request" label
3. Describe the use case and proposed solution
